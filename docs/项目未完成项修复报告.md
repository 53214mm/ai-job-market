## 二、详细修复内容

### 1. 修复 AI 简历分析 Bug（严重）

**问题**：`ResumeServiceImpl.aiAnalyze()` 调用 AI 获取分析结果后，结果字符串被丢弃，所有评分字段存储为空。

**修复**：
- 引入 Jackson `ObjectMapper` 解析 AI 返回的 JSON
- 添加 `extractJson()` 方法从 AI 回复中提取 JSON
- 解析 `overallScore`、`formatScore`、`contentScore`、`keywordScore`、`strengths`、`weaknesses`、`suggestions` 并写入数据库
- 容错处理：JSON 解析失败时保存原始结果到 `suggestions` 字段

**涉及文件**：`src/main/java/.../service/impl/ResumeServiceImpl.java`

---

### 2. 通知集成到投递流程

**问题**：`ApplicationServiceImpl` 的状态变更和面试安排不会推送通知给求职者。

**修复**：
- 注入 `NotificationService`
- `updateStatus()` 中：状态变更后自动推送 `APPLICATION_UPDATE` 类型通知
- `scheduleInterview()` 中：安排面试后自动推送 `INTERVIEW_INVITE` 类型通知
- 通知内容包含职位名和状态描述

**涉及文件**：`src/main/java/.../service/impl/ApplicationServiceImpl.java`
